## Hexo blog  

当更换PC时，只需要在新的PC上安装hexo环境，把本项目clone至本地，则可以
继续添加blog

-----

### How to Run 

* install dependencies
> npm install  
> npm i hexo-cli -g  
> npm install hexo-deployer-git --save

* serve with hot reload at localhost:4000
> hexo s  
> hexo d -g  
> hexo clean

### possible problem  

* 本地未安装node.js  
* 依赖是否全部已安装
* node版本控制在  node/11.12.0 ，过高的node版本会构建错误！！！
