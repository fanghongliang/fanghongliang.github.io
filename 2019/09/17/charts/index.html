G2引入

  1. CDN：
    <script src="https://gw.alipayobjects.com/os/lib/antv/g2/3.4.10/dist/g2.min.js"></script>

  2. NPM:
    $ npm install @antv/g2 --save
    import G2 from '@antv/g2'

  3. 本地脚本：
    <script src="./g2.js"></script>
